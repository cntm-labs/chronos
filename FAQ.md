# Frequently Asked Questions (FAQ)

## ❓ General
**Q: What is chronos?**
A: The orchestration layer that fuses symbolic math with neural networks to provide AI with verifiable memory and reasoning.

## 🛠️ Technical
**Q: How do I run tests?**
A: Use the command: `cargo test`.

**Q: Which languages are supported?**
A: Primarily `Rust (Axum), Cntm-Graph, Isotime, Mojo (CUDA), Lean`.
