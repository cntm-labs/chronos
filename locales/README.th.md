## 🎯 พันธกิจ
Chronos คือเลเยอร์การจัดการ (orchestration layer) ที่ผสานสัญลักษณ์ทางคณิตศาสตร์เข้ากับโครงข่ายประสาทเทียม เพื่อให้ AI มีหน่วยความจำและการให้เหตุผลที่ตรวจสอบได้ แก้ปัญหาการสูญเสียความจำและอาการประสาทหลอนของ AI ด้วยการสร้างเลเยอร์หน่วยความจำที่เลียนแบบสมองมนุษย์

## 🚀 คุณสมบัติหลัก
- **เครื่องมือพิสูจน์ความถูกต้อง (Lean-powered):** กราฟความรู้ที่ปราศจากข้อผิดพลาดโดยใช้คณิตศาสตร์ขั้นสูงเพื่อตรวจสอบตรรกะ
- **การติดตามความรู้ตามกาลเวลา:** การบันทึกและสืบค้นความสัมพันธ์ของความรู้ตามลำดับเวลาภายในกราฟ
- **โมเดลการรับรู้เชิงทำนาย (Mojo/CUDA):** การพยากรณ์เหตุการณ์ในอนาคตตามความสัมพันธ์ของกราฟที่ซับซ้อน
- **การบีบอัดความรู้แบบลำดับชั้น:** การจัดการบริบทมหาศาลผ่านการปรับขนาดความรู้หลายระดับ

## 🛠 เทคโนโลยีที่ใช้
- **Core:** Rust (Axum)
- **Graph:** Cntm-Graph (เครื่องมือกราฟประสิทธิภาพสูง)
- **Time-Series:** BlowTime (เครื่องมือจัดเก็บอนุกรมเวลา)
- **AI Engine:** Mojo (PyTorch พร้อม CUDA)
- **Logic:** Lean Proof Assistant
