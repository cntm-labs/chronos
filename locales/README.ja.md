> Chronosは、シンボリック数学とニューラルネットワークを融合させ、AIに検証可能なメモリと推論を提供するオーケストレーションレイヤーです。
The orchestration layer that fuses symbolic math with neural networks to provide AI with verifiable memory and reasoning.

## ✨ 特徴 (Features)
- **形式手法検証エンジン (Lean駆動):** 高度な数学を使用して論理を検証する、エラーのないナレッジグラフ。
- **時間的知識トレース:** グラフ内の時系列に沿った知識関係の記録と探索。
- **予測的認知モデル (Mojo/CUDA):** 複雑なグラフ関係に基づいた未来のイベント予測。

cargo build
