> Chronos 是将符号数学与神经网络融合的编排层，旨在为 AI 提供可验证的记忆和推理能力。
The orchestration layer that fuses symbolic math with neural networks to provide AI with verifiable memory and reasoning.

## ✨ 特性 (Features)
- **形式化验证引擎 (Lean 驱动):** 使用先进数学实现的零错误知识图谱。
- **时间知识追踪:** 在图谱中记录和遍历按时间顺序排列的知识关系。
- **预测性认知模型 (Mojo/CUDA):** 基于复杂图关系预测未来事件。

## 🛠️ 快速开始 (Quick Start)

cargo build
